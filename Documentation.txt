App documentation:
After downloading the zip file from Github head(cd in terminal/hyper) to that repository and follow the steps

>Step 1
go to browser and download node LTS (if not already installed)

>Step 2
 install npm
 Run the command= pip install npm

>Step 3
 Run the command= npm install 
This will install the dependencies

>Step 4
run the command = npm start
This will set up the server

The app will be rendered on the browser window at localhost:3000
